# OrbitOps Mobile - Ground Control Station

## Descrição da Solução
Na nova Economia Espacial, infraestruturas críticas na Terra (como agricultura, previsão de desastres e conectividade) dependem de constelações de nanossatélites. 

O *OrbitOps Mobile* atua como uma solução de telemetria e governança de campo (Ground Control). Ele conecta a infraestrutura espacial à equipa de engenharia na Terra. Através da aplicação, os engenheiros conseguem monitorizar em tempo real a temperatura térmica e a bateria dos CubeSats, além de gerir incidentes críticos (Alertas). O encerramento de um alerta impede que o satélite degrade e se torne "lixo espacial", assegurando a continuidade dos serviços terrestres que dependem desses dados.

2. Abra o terminal na pasta raiz do projeto.
3. Instale as dependências executando o comando:
   ```bash

Inicie o servidor do Expo executando:

Bash
npx expo start
